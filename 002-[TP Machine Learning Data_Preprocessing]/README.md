## Apprentissage supervisé

Diviser la BDD (base de données) entre un jeu **d'apprentissage** et un jeu de **test**.

Apprentissage : 
- `Xtrain` - Features 
- `Ytrain` - Labels

Test  :
- `Xtest` - Features
- `Ypred` - Labels
